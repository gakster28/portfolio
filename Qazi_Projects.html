The Adventures of Bob the Cat

August 2017 - September 2017, PLTW Computer Science/Software Engineering

The Adventures of Bob the Cat was a game created using Scratch. This was the first project in the PLTW Computer Science/Software Engineering class that I am taking. At first, I couldn't find a way to time the character's interactions, but then I discovered I could use Scratch's broadcasting system. It essentially released a code-word after a message was sent, so the other character knows when to speak.
